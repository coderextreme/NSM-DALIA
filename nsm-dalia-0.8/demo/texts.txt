This file shows NSM texts in NSM format.

By parsing whis file with NSM-DALIA, you will get the

THIS PERSON IS SAD =
@P
(a) this person feels something
(b) sometimes a person thinks like this
(c)     I know that something bad happened
(d)     I do not want things like this to happen
(e)     I cannot think now that I will do something because of this
(f)     I know that I cannot do anything
(g) because of this this person feels something bad
(h) this person feels something like this
@e


GOD =

there is not anyone else like this someone
this someone exists always
everything exists because this someone wants these things to exist
people exist because this someone wants people to exist
this someone exists because this someone exists
this someone exists not because of anything else
this someone lives
@e
